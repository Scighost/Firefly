# Firefly

Firefly 是一个用于在 Windows 桌面展示流萤 Live2D 模型的应用。相比于通用的 Live2D 模型查看器，Firefly 的安装包大小、内存占用、CPU/GPU 负载都维持在很低的水平，在保证观感和交互体验的同时，尽量减少对日常使用与游戏性能的影响。

Live2D 模型来源：[bilibili@是依七哒](https://space.bilibili.com/457683484)


## 下载安装

在 [Releases](https://github.com/Scighost/Firefly/releases) 页面下载安装包，Firefly 需要 Windows 10 1809 (版本 17763) 或更高版本才能运行。


## 开发指南

1. 下载 [Live2D Cubism SDK for Native](https://www.live2d.com/sdk/about/)
2. 将 SDK 中的 `Live2DCubismCore.dll` 放到 `Live2DCSharpSDK/Live2DCSharpSDK.Framework/` 文件夹中
