# Firefly

Firefly is an application for displaying flowing firefly Live2D models on the Windows desktop. Compared with general Live2D model viewers, Firefly maintains a very small installation size, low memory usage, and minimal CPU/GPU load, while ensuring visual and interactive experience and minimizing impact on everyday use and gaming performance.

Live2D model source: [bilibili@是依七哒](https://space.bilibili.com/457683484)

## Download & Installation

Download the installer from the [Releases](https://github.com/Scighost/Firefly/releases) page. Firefly requires Windows 10 version 1809 (build 17763) or later to run.

## Development Guide

1. Download [Live2D Cubism SDK for Native](https://www.live2d.com/sdk/about/).
2. Place `Live2DCubismCore.dll` from the SDK into the `Live2DCSharpSDK/Live2DCSharpSDK.Framework/` folder.
